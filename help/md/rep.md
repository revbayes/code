## name
rep
## title
Replicate a value
## description
'rep' creates a vector of 'n' copies of the value 'x'.
## details
'rep' creates a vector of 'n' elements, each with value 'x', preserving the type of 'x' in the returned vector.

Sebastian Hoehna

## example

## name
setwd
## title
Set and print the working directory
## description
Set the current working directory which RevBayes uses.
## details
## authors
Sebastian Hoehna

## example
	# get the current working directory
	getwd()
	
	# let us set a new working directory
	setwd("~/Desktop")
	
	# check the working directory again
	getwd()

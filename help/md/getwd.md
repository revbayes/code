## name
getwd
## title
Get and print the working directory
## description
Get the current working directory which RevBayes uses.
## details
## authors
Sebastian Hoehna
## see_also
setwd
## example
	# get the current working directory
	getwd()
	
	# let us set a new working directory
	setwd("~/Desktop")
	
	# check the working directory again
	getwd()
